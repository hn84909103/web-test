Overview

The kiosk user interface for the shark exhibition allows visitors to navigate the species labels through the kiosk with a simple touch. Each species page includes informative multimedia sources along with the content. The layout design is family-friendly and visually appealing to target audiences, such as families with children, school groups, and young adult couples.

Overview

The kiosk user interface for the shark exhibition allows visitors to navigate the species labels through the kiosk with a simple touch. Each species page includes informative multimedia sources along with the content. The layout design is family-friendly and visually appealing to target audiences, such as families with children, school groups, and young adult couples.

My Role

Define user requirements, wireframing, prototyping

Summary

The kiosk was planned for an aquarium exhibition, so I picked a darker-tone background with white text for easier reading in a dimly light environment.

Visually appealing to all visitors, the species info page included graphic icons, videos, and fun facts.

The goal of the kiosk UI is for users to easily navigate and learn information about the species through the interface design.
